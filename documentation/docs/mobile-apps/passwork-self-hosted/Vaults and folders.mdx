## Vaults

In the Passwork app you can create new vaults and manage existing ones by creating new folders, renaming vaults and deleting them.

</div>

:::tip
1. Select a vault and open the **Vault actions** menu
2. Select an action
:::

### Create vault

To create a vault, tap the plus button in the top right corner of the vault list, enter a name for the new vault, then save the changes:

  <img

  />
</div>

:::tip
1. Tap the plus button
2. Create a vault
:::
