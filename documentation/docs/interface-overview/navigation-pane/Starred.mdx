## Starred

**Starred** is a list of favorite passwords marked with a star label for quick access. Each Passwork user has their own list of starred passwords.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/X2f2iRELZRe_LdMiyOAKj_133-interface-overview-navigation-column-starred-1.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. On the Navigation pane, select **Starred**  
2. The list of starred passwords will be displayed on the right
:::

You can add or remove passwords from your personal Starred list by clicking the **Star icon** in the Password card.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/nGAAWnv2rm39MDygtjIMm_133-interface-overview-navigation-column-starred-3.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Click the Star icon to add a password to **Starred**
:::

Additionally, you can add passwords to Starred by clicking the **Star icon** while browsing through vaults and folders.

<div style={{ textAlign: 'center' }}>

:::tip
1. On the Content pane, click the Star icon to add a password to **Starred**
:::
