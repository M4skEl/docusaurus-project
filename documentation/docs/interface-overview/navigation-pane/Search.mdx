## Use the **Search bar** to quickly find the password you need. Enter text or filter by any colors or tags and Passwork will display relevant results.

<div style={{ textAlign: 'center' }}>

    src="/img/assets/Bo60iwlZxKI4YEAIjJgML_131-interface-overview-navigation-column-search-bar.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Type your text into the **Search bar**  
2. Choose **color labels**  
3. Choose **tags**  
4. View the **Search results**
:::

:::info
As Passwork stores data in an encrypted format, a regular search functionality won't work. To perform a fast search Passwork builds its own secured indexes for each password. You can configure search in the System settings.
:::
