Администраторы Пассворка и пользователи с соответствующими правами могут просматривать историю авторизаций других пользователей.

В окне истории авторизаций указывается дата входа, идентификатор устройства и его IP-адрес. Помимо этого, здесь можно завершить все сессии пользователя.

Для просмотра авторизаций пользователя перейдите на его страницу и выберите пункт **История авторизаций** в панели справа:
