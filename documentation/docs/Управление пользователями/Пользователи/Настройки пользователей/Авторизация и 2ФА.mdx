## Сброс пароля

:::danger
При сбросе мастер-пароля, пользователь будет отключен от всех сейфов
:::

Если пользователь потерял или забыл свой локальный пароль или мастер-пароль, его всегда можно сбросить. После сброса будет сгенерирован временный пароль, который позволит пользователю войти в Пассворк и назначить новый постоянный пароль.

После нажатия кнопки **Сбросить**, во всплывающем окне будет показан соответствующий временный пароль, который потребуется сменить при входе в Пассворк.
