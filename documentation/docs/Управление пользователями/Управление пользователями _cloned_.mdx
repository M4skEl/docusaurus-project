Раздел **Управление пользователями** позволяет легко изменять настройки, устанавливать методы авторизации, управлять ролями и приглашать пользователей в Пассворк. Чтобы открыть раздел управления пользователями, перейдите в меню **Настройки и пользователи**.

Страница **управления пользователями** разделена на три раздела: **пользователи**, **роли** и **приглашения**.

## Пользователи

Эта вкладка содержит список всех пользователей вашего Пассворка, их роли, уровни доступа и типы авторизации. Справа от списка пользователей вы найдете инструменты для создания, редактирования, приглашения и удаления пользователей. Вы также можете назначать пользователям роли, снимать с них роли и настраивать определенные типы авторизации.

:::tip
1. Находите пользователей с помощью поисковых запросов и фильтров  
2. Просматривайте список всех пользователей, их роли, статусы и типы авторизации  
3. Создавайте или приглашайте новых пользователей  
4. Управляйте ролями, методами авторизации и удаляйте пользователей
:::

:::info
В Пассворке версии 6.0.2 и выше вы можете предотвратить изменения пользователей во время синхронизации LDAP с помощью кнопки **Синхронизации LDAP** на правой панели
:::

## Роли

Роли — это эффективный инструмент управления пользователями, позволяющий настраивать права доступа к сейфам и папкам. Вы можете создать неограниченное количество ролей и применить их к любому количеству пользователей. Пользователь может иметь несколько ролей, которые можно включать и отключать одним нажатием клавиши.

2. Просматривайте список всех ролей, количество связанных с ними сейфов и пользователей, а также отключайте и удаляйте роли  
3. Создавайте новые роли
:::

:::info
Вы можете сопоставить группы **Active Directory** с ролями. Для каждой группы AD можно указать одну или несколько ролей. Так пользователи из определенной группы будут автоматически получать нужные роли.
:::

### Отключение прав роли

Переключатель **Отключения прав роли** в списке ролей позволяет временно отключить все права доступа определенной роли.

:::info
Все пользователи с деактивированной ролью автоматически потеряют доступ к сейфам и папкам, связанным с этой ролью, если только у них нет другой роли, предоставляющей такой доступ.
:::

## Приглашения

Приглашения представляют из себя сгенерированные коды, которые можно отправлять незарегистрированным пользователям.

При создании кода вы можете предварительно назначить пользователю роли, а также установить дату истечения срока действия приглашения. Это позволяет пользователям начать пользоваться Пассворком сразу после регистрации, не дожидаясь одобрения администратора.
