Passwork makes it possible to view folder access rights using several tools. All of them you can find on the **Folder control panel**.

## Folder access

The **Folder access window** offers a convenient way to view lists of users and roles that have access to a selected folder. You can also add, delete, or modify access levels for these users and roles.

## Manage user access rights

In the Folder access window, you can add or remove users from the selected folder, easily find the necessary users and change their access rights.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/d2imRMLsfj2HH_4KjH54J_348-vaults-and-folders-actions-with-folders-view-folder-access-rights-4.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Go to the **Users** tab  
2. Add more users if needed  
3. Search for users or filter them by access rights  
4. Select users and modify their access rights  
5. Click **Save** to keep your changes or **Cancel changes** to discard them
:::

You can select multiple users at once using the checkboxes to the left of their names and change or remove access rights for all of them simultaneously.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/hyNWXpBhJKfnwSUBJWQ8a_348-vaults-and-folders-actions-with-folders-view-folder-access-rights-5.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Select one or more users  
2. Go to the **Choose action** dropdown menu  
3. Select the desired action  
4. Click **Save** to apply the changes
:::

### Manage role access rights

In the Folder access window, you can add or remove roles from the selected folder, find the necessary roles and change their access rights.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/qzDupNWwlph1EwNL0--bh_348-vaults-and-folders-actions-with-folders-view-folder-access-rights-6.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Go to the **Roles** tab  
2. Add more roles if needed  
3. Search for roles or filter them by access rights  
4. Select a role and change its access rights  
5. Click **Save** to keep your changes or **Cancel changes** to discard them
:::

You have the option to select multiple roles at once using the checkboxes to the left of their names and change or remove access rights for all of them simultaneously.
