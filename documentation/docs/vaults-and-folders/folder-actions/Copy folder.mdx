Copying a folder automatically duplicates the folder and all its contents to another vault or folder, including user access settings.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/iGXZKA77j0IVgW4jZWcoV_345-vaults-and-folders-actions-with-folders-copy-folder-1.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. On the Navigation pane, select the folder you want to copy  
2. Go to the **Folder settings** menu  
3. Select **Copy**  
:::

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/S_FJBdGVbCyUqz9EJ275m_345-vaults-and-folders-actions-with-folders-copy-folder-2.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. In the dropdown list, choose one or more folders or vaults you want to copy the selected folder to  
:::

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/GQqPObf-UqRA8Uj2q4spr_345-vaults-and-folders-actions-with-folders-copy-folder-3.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Check the list of vaults and folders where the copy will be sent to  
2. Click **Copy** to apply changes  
:::

:::info
Another way to copy a folder is by hovering your mouse over the folder icon on the Content pane and dragging it to another location on either the Content or the Navigation pane. You can also select multiple folders or passwords simultaneously and copy them together if needed.
:::
