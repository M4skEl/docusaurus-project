When you delete a folder, it will be moved along with all its contents to the [Interface overview/Navigation pane/Bin](../../interface-overview/navigation-pane/Bin.mdx).

    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. On the Navigation pane, select a folder you want to delete  
2. Go to the **Folder settings** menu  
3. Select **Delete**  
:::

In the pop-up window, confirm the deletion.
