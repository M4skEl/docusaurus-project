Moving a folder transfers it to another vault or folder with all its contents.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/7orgwnQlVCinR6tBscA3N_344-vaults-and-folders-actions-with-folders-move-folder-1.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. On the Navigation pane, select a folder you want to move  
2. Go to the **Folder settings** menu  
3. Select **Move**  
:::

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/Mf1WPoqt61U-lPqk-UTlT_344-vaults-and-folders-actions-with-folders-move-folder-2.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. In the dropdown list, choose a destination for the selected folder  
:::

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/tZ1B6cKuGf1IKh9LxGXdt_344-vaults-and-folders-actions-with-folders-move-folder-3.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Check the destination of the selected folder  
2. Click **Move** to apply the changes  
:::

:::info
You can also move a folder by hovering over the folder icon on the Content pane and dragging it to another location. It's possible to select multiple folders or passwords by holding Ctrl and clicking on them and move them together.
:::
