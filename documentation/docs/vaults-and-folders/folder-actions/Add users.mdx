Adding users to a folder is similar to adding them to a vault, but users will only have access to the passwords stored in that specific folder.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/G6C7rsj4XjOEo2UqSfB-g_343-vaults-and-folders-actions-with-folders-add-users-to-folder-1.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. On the Navigation pane, select a folder  
2. Click the **Folder access** button
:::

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/WEijggVpPKh1A7LdLK7LU_343-vaults-and-folders-actions-with-folders-add-users-to-folder-2.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. In the pop-up window click **Add user** or review access settings
:::

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/sHS46V3g_nRRdwQetgs3j_343-vaults-and-folders-actions-with-folders-add-users-3.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. In the dropdown list, select a user and set their access level to the folder  
2. Add more users if needed  
3. Click **Confirm** to save the changes
:::
