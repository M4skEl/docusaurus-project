Use folders to establish a multi-level structure within your vaults.

:::tip
1. Select a **vault**  
2. Go to the **Vault settings** menu  
3. Select **Create folder**  
:::

In the opened window, enter a name for the new folder, check the access rights and save the changes.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/A9kUdo3ST_tqLKGtbXgsL_331-vaults-and-folders-actions-with-vaults-create-folder-2.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Enter a name for the folder  
2. Click **Save** to apply the changes  
:::

:::info
New folders inherit the access levels of the parent vault. When creating a folder, you will be able to see which users and roles will have access to it.
:::
