:::danger
Be cautious when deleting vaults, as this action is irreversible and will permanently delete all the passwords contained within them.
:::

You can delete vaults if you have administrative access to them. Passwords and folders which were moved to the Bin from a selected vault will also be erased.

:::tip
1. Select a vault you want to delete  
2. Go to the **Vault settings** menu  
3. Select **Delete**
:::

In the pop-up window, confirm the deletion.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/Dzx4wEGqh_P6PnmAp8Dpx_339-vaults-and-folders-actions-with-vaults-delete-vault-2.png"

    style={{ display: 'block', margin: '0 auto' }}

</div>

:::tip
1. Enter the name of the vault you want to delete  
2. Click **Delete vault**
:::
