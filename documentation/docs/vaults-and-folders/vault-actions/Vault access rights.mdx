In Passwork you can manage vault access rights using several tools. All of them can be found on the **Vault control panel**.

It contains the details about users and passwords connected to the vault. Click on any element of the **Vault control panel** to open the corresponding access settings:

## Manage user access

In the Vault access window, you can add or remove users from the vault, search through the user list and manage the access rights.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/erfY6XIAgL7CzA22GM_vS_3361-vaults-and-folders-actions-with-vaults-vault-access-rights-4.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Go to the **Users** tab  
2. Add more users if needed  
3. Search for users or filter them by access rights  
4. Select users and modify their permissions  
5. Click **Save** to apply the changes  
:::

## Manage role access

In the Vault access window, you can add or remove roles from the selected vault, find the necessary roles and change their access rights.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/OcsPxp_Zc5MxvxDr9Q6Ov_3361-vaults-and-folders-actions-with-vaults-vault-access-rights-6.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Go to the **Roles** tab  
2. Add more roles if needed  
3. Search for roles or filter them by access rights  
4. Select roles and modify their permissions  
5. Click **Save** to apply the changes  
:::

## Additional access dashboard

In the **Additional access** window you can view all shortcuts and password links created within a selected vault. It also provides details about all passwords that users of the vault have sent to others.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/DoFvagpgk50S7K9vuXEvD_3361-vaults-and-folders-actions-with-vaults-vault-access-rights-8.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Select a vault  
2. On the **Vault control panel**, click on the list of sent passwords, links and shortcuts  
:::

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/cidSWmMmOWrIAtUaH7SIF_3361-vaults-and-folders-actions-with-vaults-vault-access-rights-10.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. **Sent passwords**, **Links** and **Shortcuts** tabs  
2. Find sent passwords, links or shortcuts using the search  
3. Select a password, link, shortcut, or group of entries with checkboxes and manage access rights  
4. Click **Save** to apply the changes  
:::
