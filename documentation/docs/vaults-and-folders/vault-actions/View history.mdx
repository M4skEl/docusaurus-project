All changes to passwords, folders, role assignments, access management, and other activities, both within private or organization vaults can be tracked in the **Activity log**. You can access the Activity log from the **Vault settings** dropdown menu:

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/rpZ9q8A73ZIBDiNlcrY3U_337-vaults-and-folders-actions-with-vaults-activity-log-1.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. On the Navigation pane, select a vault  
2. Go to the **Vault settings** menu  
3. Select **History**
:::

Or by going to the **Settings and Users** menu and opening the **Activity log** there:

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/uLdlVthmu_ayt4s3c94Hd_337-vaults-and-folders-actions-with-vaults-activity-log-2.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Event history of the selected vault
:::

You can also see your last action in the selected vault by hovering over your profile image on the **Vault control panel**.
