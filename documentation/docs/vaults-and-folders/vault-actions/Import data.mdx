Passwork lets you quickly import passwords kept in **JSON**, **KeePass XML** and **CSV** formats into a specific vault, folder, or into the Passwork root directory. JSON and XML files are used to transfer structured information, while CSV stores raw data as plain text.

To set up the import, go to the settings menu of the vault or folder, or go to the **Import data** section in **Settings and users**:

## Passwork JSON and KeePass XML

**Passwork JSON** and **KeePass XML** formats are able to store data exported from Passwork or KeePass while preserving the original vault and folder structure, making it easy to import.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/k6LkXttYtBvxXZ1qZnzdA_511-import-and-export-import-3.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Select **Passwork JSON** or **KeePass XML** format  
2. Choose the files  
:::

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/udA8YUWVaRQnYyY1rL-y9_511-import-and-export-import-4.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Select vaults and folders you want to import  
2. Click **Next** to continue  
:::

Files in Passwork JSON and KeePass XML formats preserve the document structure. When choosing a target location, you will be able to:

- **Save structure and import to a specific location**: All selected vaults will be turned into folders inside the target vault or folder. Folder structure will be restored and all passwords will remain in their original folders.
- **Ignore initial structure and import to a specific location**: All passwords will be saved to a specific vault or folder and no structure will be recreated.
- **Save structure and import to the root of Passwork**: All selected vaults and their structure will be recreated. Passwords will be saved in the folders they were exported from.

:::tip
1. Choose the import option  
2. Select the destination directory  
3. **Import passwords**  
:::

## CSV

You can upload your existing data to Passwork from a CSV file, or paste it manually. For CSV files, no preparation is needed. Simply upload the file and Passwork will suggest columns from which data can be imported.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/UjH1XQyF8DkafPJwvH_8L_512-import-and-export-import-csv-1.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Upload a file or paste the data  
2. Choose a delimiter or **Skip the first line** if needed  
3. Click **Next**  
:::

In the next window, select which fields in Passwork will correspond to the existing values. You can add as many fields as you need using the **Add additional field** button:

  />
</div>

:::tip
1. Match the fields in Passwork with the values  
2. Add additional fields if needed  
3. Click **Next**  
:::
