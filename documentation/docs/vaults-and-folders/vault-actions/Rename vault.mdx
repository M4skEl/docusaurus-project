:::info
Only administrators can rename a vault
:::

:::tip
1. Select a vault  
2. Go to the **Vault settings** menu  
3. Select **Rename**
:::

In the opened window, enter a new name for the vault and save the changes.

:::tip
1. Enter a new name for the vault  
2. Click **Save** to apply the changes
:::
