## Сохранение пароля

Расширение Пассворка автоматически распознает формы авторизации и предлагает сохранить или использовать учетные данные, связанные с веб-страницей.

При первом входе на сайт расширение предложит вам сохранить пароль:

<div style={{ textAlign: 'center' }}>
  <img

    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Выберите конечное расположение пароля  
2. Поля **Логин**, **Пароль** и **URL** будут заполнены автоматически, при необходимости заполните остальные поля вручную  
3. Нажмите **Сохранить**, чтобы применить изменения
:::

## Автозаполнение пароля

При последующем посещении страницы, с которой связан сохраненный пароль, Пассворк предложит вам автоматически заполнить поля для входа в аккаунт:

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/QKZtfOnnubSbfN3D1EPqb-YfGEeXs7kOGh9FAPknS7p-20240925-152441.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Щелкните по предложенному паролю, чтобы использовать его для авторизации  
2. Нажмите **Никогда для этого сайта**, чтобы отключить автозаполнение для текущего сайта и добавить его в [Игнорируемые](./../Настройки%20расширения.mdx)
:::

:::info
Вы можете автоматически заполнять формы авторизации, щелкнув правой кнопкой мыши по полю формы и выбрав вход через Пассворк в контекстном меню
:::
