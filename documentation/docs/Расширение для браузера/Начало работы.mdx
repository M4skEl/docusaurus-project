# О расширении

Браузерное расширение Пассворка — это полноценный менеджер паролей, позволяющий:

- Пользоваться своими паролями непосредственно через расширение
- Автоматически заполнять формы авторизации
- Сохранять и редактировать пароли
- Искать по имени, цветам и тегам
- Генерировать новые пароли
- Защитить данные с помощью PIN-кода

## Установка расширения

:::danger
Использование расширения доступно для пользователей с [соответствующим правом](./../Управление пользователями/Пользователи/Настройки пользователей/Права и настройки.mdx)
:::

Расширение Пассворка доступно для <a href="https://apps.apple.com/app/passwork-extension/id1584570476" target="_blank">Safari</a>, <a href="https://chrome.google.com/webstore/detail/passwork-self-hosted/ibiipnmmlnehmeonnhbdajcfagcgihkl" target="_blank">Google Chrome</a>, <a href="https://addons.mozilla.org/en-US/firefox/addon/passwork-self-hosted/" target="_blank">Mozilla Firefox</a> и <a href="https://microsoftedge.microsoft.com/addons/detail/passwork-selfhosted/nakajpankdpmmaaljmdjmbnnicaolcpc" target="_blank">Microsoft Edge</a>.

Чтобы установить расширение, перейдите в магазин дополнений вашего браузера по одной из ссылок выше. После того, как вы загрузили и установили расширение, подключите его к своему Пассворку, введя адрес хоста:

## Отключение встроенного менеджера паролей

Большинство современных браузеров использует собственные менеджеры паролей, которые часто мешают работе сторонних инструментов. Поэтому мы рекомендуем отключить встроенный менеджер вашего браузера перед началом работы с расширением Пассворка.

### Google Chrome

1. Перейдите в менеджер паролей Google, открыв карточку профиля в правом верхнем углу и нажав на **иконку ключа:**

   <div style={{ textAlign: 'center' }}>
     <img
       src="/img/assets/QKZtfOnnubSbfN3D1EPqb-kJ85roEk4KnnrpYEOPxSE-20240924-135723.png"
       width="600"
       style={{ display: 'block', margin: '0 auto' }}
     />
   </div>

2. Откройте вкладку **Настройки** слева и отключите все функции управления паролями:

   <div style={{ textAlign: 'center' }}>
     <img
       src="/img/assets/QKZtfOnnubSbfN3D1EPqb-cN_JI_fYLvO02y8Y83YkZ-20240924-141552.png"
       width="600"
       style={{ display: 'block', margin: '0 auto' }}
     />
   </div>
