API Пассворка позволяет принимать, создавать и обновлять пароли, папки и сейфы, управлять пользователями и настройками системы. Это удобный способ интегрировать Пассворк в вашу инфраструктуру. Вы также можете воспользоваться нашим [Python-коннектор](./../../API и интеграции/Python-коннектор.mdx) для упрощения интеграции.

---

На странице Авторизация и 2ФА нажмите **Сгенерировать токены.**

Генерация токенов доступна в течение пяти минут с момента авторизации.

<div style={{ textAlign: 'center' }}>
  <img
    src="https://archbee-image-uploads.s3.amazonaws.com/QKZtfOnnubSbfN3D1EPqb-yOs_1uKpbfqpuOyDPg5Id-20250429-093802.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

---

Скопируйте ваши токены из всплывающего окна.

Действия, совершаемые с использованием токенов, будут выполняться от имени пользователя, которому они принадлежат.
