**Избранные** — это список паролей, отмеченных звездочкой для быстрого доступа. Каждый пользователь Пассворка имеет свой собственный список избранных паролей.

:::tip
1. Выберите **Избранные** в колонке навигации  
2. Список избранных паролей появится в окне содержимого
:::

Вы можете добавить или удалить пароль из своего списка избранного, кликнув по звездочке в карточке пароля.
