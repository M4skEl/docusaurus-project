**Общие сейфы** — основной тип сейфов в Пассворке. Они подходят для большинства задач: вы можете создавать хранилища для разных проектов, отделов и других подразделений вашей компании. Общий сейф создается при добавлении других пользователей в **приватный сейф**. Сейфы организации доступны администраторам Пассворк и пользователям с соответствующими правами доступа.

Пароли в хранилищах можно упорядочить, поместив их в папки. Вы можете использовать неограниченное количество вложенных папок для создания гибкой многоуровневой структуры.
