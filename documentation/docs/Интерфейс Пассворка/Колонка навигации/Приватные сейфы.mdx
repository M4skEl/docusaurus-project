**Приватные сейфы** — персональные сейфы, которые недоступны системным администраторам и другим пользователям без разрешения их владельца. Владельцы могут приглашать других пользователей в свои личные сейфы и предоставлять им соответствующие права доступа.

:::info
По умолчанию все сейфы в Пассворке создаются приватными
:::
