Пассворк позволяет быстро экспортировать пароли в файлы формата **CSV** и **JSON**. Формат JSON полностью сохраняет структуру сейфов и папок, в то время как CSV хранит исходные данные в виде обычного текста.

1. Чтобы воспользоваться экспортом, откройте меню **Настроек сейфа** и нажмите **Экспорт данных**:

   <div style={{ textAlign: 'center' }}>
     <img
       src="/img/assets/QKZtfOnnubSbfN3D1EPqb-PddWhmaE2E3OK8pBgOD1A-20250403-095251.png"
       width="600"
       style={{ display: 'block', margin: '0 auto' }}
     />
   </div>

2. В появившемся окне отметьте сейфы и папки, содержимое которых вы хотите экспортировать и выберите формат экспорта:

   <div style={{ textAlign: 'center' }}>
     <img
       src="/img/assets/QKZtfOnnubSbfN3D1EPqb-nikF4qs19xRj6FMrwGjQt-20250317-095342.png"
       width="600"
       style={{ display: 'block', margin: '0 auto' }}
     />
   </div>

:::danger
Любые прикрепленные к паролям файлы не сохраняются при экспорте
:::
