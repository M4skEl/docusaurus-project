Пассворк имеет несколько инструментов для управления правами доступа к сейфу. Все они находятся на **панели управления сейфом**.

Она содержит подробную информацию о пользователях и паролях, связанных с сейфом. Щелкните на нужном элементе панели управления сейфом, чтобы открыть соответствующие настройки доступа:

## Управление доступом пользователей

В окне **доступа к сейфу** вы можете добавлять или удалять пользователей из сейфа, искать их в списке и управлять их правами доступа.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/QKZtfOnnubSbfN3D1EPqb-VqslP_duPbeYrGh3TuGPu-20250404-110807.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Откройте вкладку **Пользователи**  
2. Добавьте одного или нескольких пользователей  
3. Ищите среди пользователей или фильтруйте по уровням доступа  
4. Выберите пользователей и измените их уровень доступа
:::

### Управление доступом групп

В окне **доступа к сейфу** вы можете добавить или удалить группу из сейфа, найти необходимые группы и изменить их права доступа.

  />

:::tip

2. Добавьте одну или несколько групп  
3. Ищите среди групп или фильтруйте по уровням доступа  
4. Выберите группы и измените их уровень доступа
:::

### Управление запросами

Если в вашем Пассворке включен режим клиентского шифрования, то при добавлении в группу нового пользователя, администратору сейфа потребуется подтвердить запрос в соответствующей вкладке Настроек доступа. Чтобы увидеть новые запросы, нажмите на красную иконку рядом с названием сейфа:
