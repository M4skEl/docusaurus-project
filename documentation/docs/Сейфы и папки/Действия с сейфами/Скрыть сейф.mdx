:::info
Сейфы, скрытые вами, все еще видны остальным пользователям
:::

Чтобы сделать навигацию проще, Пассворк позволяет скрывать сейфы из списка сейфов.

Откройте выпадающее меню **Настроек сейфа** и нажмите **Скрыть**, после чего подтвердите действие во всплывающем окне.
