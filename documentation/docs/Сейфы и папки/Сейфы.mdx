Сейфы Пассворка — это зашифрованные контейнеры корневого уровня, в которых хранятся папки и пароли. Сейфы используют специальный криптографический алгоритм, который обеспечивает сохранность ваших данных.

В Пассворке есть два типа сейфов: **приватные сейфы** и **общие сейфы**.

**Приватные сейфы** — персональные хранилища, доступ к которым есть только у их владельца. Администраторы и другие пользователи не могут просматривать их содержимое без разрешения. По умолчанию каждый новый сейф создаётся приватным. Владелец сейфа может пригласить других пользователей и настроить для них права доступа.

:::info
Приватный сейф становится общим при добавлении в него других пользователей
:::

**Общие сейфы** — основной инструмент для совместной работы в Пассворке. Используйте их, чтобы хранить и делиться доступами внутри команды, отдела или между проектами.

Оба типа сейфов поддерживают многоуровневую структуру на основе папок: администраторы могут выстраивать иерархию сейфов с вложенными в них папками.
