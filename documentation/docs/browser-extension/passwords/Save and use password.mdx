# Save password

**Passwork extension** automatically recognizes login forms and prompts to save or use the credentials associated with the webpage.

When logging into a website for the first time, the extension will prompt you to save the password.

<div style={{ textAlign: 'center' }}>
  <img

    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Choose the directory where you want to save the password  
2. The **Login**, **Password**, and **URL** fields will get filled automatically, manually complete the remaining fields if needed  
3. Click **Save** to apply the changes
:::

## Autofill password

Next time you visit a page that has a saved password associated with it, Passwork will prompt you to autocomplete the sign-in fields:

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/QKZtfOnnubSbfN3D1EPqb-xD_tEnTKdp_BciiuS_Iwz-20240912-112431.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Click on a suggested password to use it in the login form  
2. Click **Never for this site** to disable autofill for the current website and send it to Ignored
:::

:::info
You can autofill login data by right-clicking on the field of a login form and choosing **Sign in using Passwork** in the context menu
:::
