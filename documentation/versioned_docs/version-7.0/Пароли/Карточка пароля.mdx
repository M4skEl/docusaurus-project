Пассворк был создан для удобного и безопасного хранения ваших паролей и других учетных данных.

Все данные о паролях представлены в виде **карточек паролей**. Она содержит всю информацию о конкретном пароле, включая его предыдущие версии и историю действий. Карточка пароля также содержит все инструменты для управления паролем.

### Избранные пароли

**Избранные** — это список паролей, отмеченных звездочкой для быстрого доступа. Каждый пользователь Пассворка имеет свой собственный список избранных паролей. Вы можете добавить пароль в избранное, нажав на **значок звездочки** справа от названия пароля. Подробнее о списке избранных паролей можно узнать на странице [Избранные](./../Интерфейс Пассворка/Колонка навигации/Избранные.mdx).
