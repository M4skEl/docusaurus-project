Используйте **строку поиска**, чтобы быстро найти нужный вам пароль. Введите текст или отфильтруйте пароли по цветам или тегам, и Пассворк покажет соответствующие результаты.

:::tip
1. Введите запрос в **строку поиска**  
2. Выберите **цветовую метку**  
3. Выберите **теги**  
4. Просмотрите **результаты поиска**
:::

:::info
Поскольку Пассворк хранит данные в зашифрованном виде, обычные методы поиска не применимы. В целях быстрого поиска Пассворк создает собственные защищенные индексы для каждого пароля. Настроить поиск можно в **Системных настройках.**
:::
