**Недавние** — это список последних использованных паролей, расположенных в хронологическом порядке. Они отображаются в области просмотра при входе в Пассворк, а также доступны в Колонке навигации.

Пароль автоматически добавляется в **Недавние** после его просмотра. Если пользователь больше не имеет доступа к паролю, он исчезнет из его списка.
