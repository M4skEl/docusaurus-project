Вы можете временно отключить все права доступа определенных групп, используя выпадающее меню в списке групп.

Все пользователи из деактивированной группы автоматически потеряют доступ к сейфам и папкам, связанным с этой ролью, если только у них нет другой роли, предоставляющей такой доступ.
