:::info
**Группы** — это эффективный инструмент управления пользователями, позволяющий настраивать права доступа к сейфам и папкам
:::

Администраторы Пассворка и пользователи с соответствующими правами могут создавать группы.

1. Чтобы создать новую группу, откройте вкладку **Группы** в разделе **Управление пользователями** и нажмите **Создать:**

       width="600"
       style={{ display: 'block', margin: '0 auto' }}
     />
   </div>

2. Во всплывающем окне введите название новой группы и сохраните изменения:
