:::info
В Пассворк 7 **роли** объединяют в себе права администрирования, статусы и пользовательские настройки из предыдущих версий.
:::

Пользователям Пассворка могут быть присвоены три базовые роли:

- **Владелец** создается во время установки Пассворка. Владелец может управлять пользователями, ролями и настройками. Владелец может быть только один и не может быть сменен или удален администраторами.
- **Администратор** назначается владельцем и по умолчанию имеет доступ к тем же функциям, что Владелец.
- **Сотрудник** — это обычный пользователь с доступом к определенным сейфам в организации, предоставляемым администраторами сейфа. При этом сотрудник может иметь права в определенном сейфе.

Помимо базовых ролей выше, вы можете создать неограниченное количество ролей со своими правами и настройками.

## Сменить роль

Чтобы изменить роль пользователя, перейдите во вкладку **Пользователи** раздела **Управление пользователями**, выберите пользователя, роль которого вы хотите изменить и выберите новую из выпадающего меню:

## Назначить владельцем

Владелец организации может назначить другого пользователя владельцем вместо себя. Для этого выберите пункт **Назначить владельцем** в выпадающем меню выбора роли.
