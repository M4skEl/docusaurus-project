Если пользователь потерял или забыл свой пароль авторизации, его всегда можно сбросить. После сброса будет сгенерирован временный пароль, который позволит пользователю войти в Пассворк и назначить новый постоянный пароль.

Чтобы сбросить пароль авторизации пользователя, выберите нужного пользователя и нажмите **Авторизация и 2ФА** в панели справа:
