Откройте **Настройки** расширения, нажав на значок **шестеренки** на панели действий.

## Общие настройки

На вкладке **Общие** вы можете изменить язык и внешний вид расширения, а также настроить функцию автоблокировки для повышения безопасности данных.

### Блокировка расширения

Для того, чтобы повысить безопасность своих данных, вы можете установить PIN-код для входа в расширение. Введите 4-значный код, чтобы настроить функцию блокировки. Вы можете дополнительно включить автоматическую блокировку по истечении определенного времени.

## Игнорируемые сайты

Вкладка **Игнорируемые сайты** содержит список сайтов, которые вы добавили в Игнорируемые через всплывающее окно [сохранения и использования пароля](./Пароли/Сохранить и использовать пароль.mdx). Для таких сайтов Пассворк не будет сохранять или автоматически заполнять пароли.
