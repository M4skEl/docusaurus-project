To open the **app settings**, tap on your profile picture in the top-left corner of the app. Here you can change your PIN code, set the inactivity period, change language and appearance and log out of the app.

## API Settings {#api-settings}

To use the Passwork mobile app, you need to enable the Passwork API in your Passwork instance settings. This allows the mobile app to communicate with your Passwork server securely.
