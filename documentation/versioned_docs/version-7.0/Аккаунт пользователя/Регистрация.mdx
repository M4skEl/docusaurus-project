## Чтобы зарегистрироваться в Пассворке, перейдите на страницу регистрации по ссылке, предоставленной вашим администратором. После ввода логина, пароля и электронной почты нажмите кнопку **Зарегистрироваться**, чтобы создать аккаунт.

В зависимости от настроек Пассворка, может потребоваться подтверждение регистрации. Если у вас есть код приглашения, вы сможете ввести его при первом входе в Пассворк.

Если в вашем Пассворке включён режим клиентского шифрования, то после первоначальной регистрации учетной записи потребуется установить мастер-пароль.
