The Passwork browser extension is a full-fledged password management application that includes the following features:

- Access to all your passwords right from the extension
- Automatic data saving and form filling
- Adding and editing passwords
- Search by name, colors, and tags
- Flexible password generator
- Extension lock with a PIN code
