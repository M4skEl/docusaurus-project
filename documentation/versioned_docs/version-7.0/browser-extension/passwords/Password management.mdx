## Add password

To create a new password, click the blue **Plus** button in the top-right corner:

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/QKZtfOnnubSbfN3D1EPqb-3MSpUOF02ylZpy0M2NzM_-20240912-094242.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

Choose a folder or vault you want to save the new password to, then fill in the required fields or use the **Password Generator** to create a unique password. The password will inherit the permissions set for the vault or folder you add it into.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/QKZtfOnnubSbfN3D1EPqb-6rKElaBHBFIqSv7kgUgXu-20240912-093431.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Select a directory for the new password
2. Use the Password generator to create a strong password
3. Automatically fill the URL field with the current page address
:::

After that, click **Save** to apply the changes

## Password actions

Click the three dots in the password card to open the **Password actions** menu. Here you can [edit](#) a password, [move, copy](#) and [delete](#) it.

### Edit

To edit a password, open the **Password actions** menu, click **Edit** and modify the necessary fields.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/QKZtfOnnubSbfN3D1EPqb-hQhzDMYWVKCy-a-56kRoE-20240912-095726.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Edit the password details
2. Click **Save** to apply the changes
:::

### Move and copy password

Choose **Move** if you want to change the location the password is stored at, or choose **Copy** if you want to create a copy of the password in a different directory.

### Delete password

To delete a password, click **Delete** in the **Password actions** menu, then confirm the action in the pop-up window.
