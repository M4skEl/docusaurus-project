Используйте папки для создания многоуровневой структуры в сейфах.

Новые папки наследуют уровни доступа сейфа. При создании папки вы сможете увидеть, какие пользователи и роли будут иметь к ней доступ.

1. Откройте меню **настроек сейфа** или **папки** и нажмите **Создать папку**:
