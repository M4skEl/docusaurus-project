При необходимости вы можете покинуть сейф, в котором состоите. Если пользователь покинул сейф, он сможет вернуться только после повторного добавления администратором.

Откройте выпадающее меню **Настроек сейфа** и нажмите **Покинуть**, после чего подтвердите действие во всплывающем окне.
