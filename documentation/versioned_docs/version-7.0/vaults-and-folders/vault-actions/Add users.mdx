Administrators can invite users to a vault and grant them access to passwords stored within.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/8zYCZOsoGfsST1AwZcKla_333-vaults-and-folders-actions-with-vaults-add-users-1.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. Select a vault  
2. Click the **Vault access** button
:::

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/lksPF_k0jq8m6kS47b1Lq_333-vaults-and-folders-actions-with-vaults-add-users-2.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. In the pop-up window, click **Add user**
:::

Here you can view existing users, add more and manage the necessary access rights.

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/a_hIWV5CsfbRqiif6IW7e_333-vaults-and-folders-actions-with-vaults-add-users-3.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. In the dropdown list, select a user and set their access level  
2. Add more users if needed  
3. Click **Confirm** to save the changes
:::
