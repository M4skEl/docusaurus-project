Passwork lets you quickly export passwords into **CSV** and **JSON** files. JSON format preserves the entire structure of vaults and folders, while CSV stores raw data as plain text.

To set up the export, go to the settings menu of the vault or folder, or go to the **Export data** section in Settings and Users.

:::tip
1. Select the vault from which you want to export data  
2. Go to the **Vault settings** menu  
3. Select **Export data**  
:::

:::tip
1. Select the format for exporting data  
2. Select vaults and folders you want to export  
3. Click **Export**  
:::

:::danger
Any file attachments linked to passwords won't be included in the exported data
:::

:::info
Administrator can restrict access to import and export features for specific users and user groups
:::
