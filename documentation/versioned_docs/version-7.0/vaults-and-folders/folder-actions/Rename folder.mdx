:::info
Only administrators and users with granted administrative rights can rename a folder
:::

:::tip
1. Select a folder you want to rename  
2. Go to the **Folder settings** menu  
3. Choose **Rename**
:::

<div style={{ textAlign: 'center' }}>
  <img
    src="/img/assets/5sqm86xEnC8UOs4Dl-nfB_342-vaults-and-folders-actions-with-folders-rename-folder-2.png"
    width="600"
    style={{ display: 'block', margin: '0 auto' }}
  />
</div>

:::tip
1. In the pop-up window, enter a new name for the folder  
2. Click **Save** to apply the changes
:::
